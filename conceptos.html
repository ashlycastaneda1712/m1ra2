Algunos conceptos:

1. Fundamentos del Desarrollo Web

Es el esqueleto de la web. Define la estructura básica (títulos, párrafos, imágenes). No es un lenguaje de programación, sino de etiquetas.

Es el diseño y la estética. Se encarga de los colores, fuentes y de que todo se vea bonito.

Es el motor. Le da interactividad a la página (botones que despliegan menús, animaciones, actualizaciones de datos sin recargar).

La parte de la web que el usuario ve y con la que interactúa.

La lógica del servidor y el manejo de datos que ocurre "detrás de cámaras".

2. Diseño y Experiencia de Usuario

Una técnica para que la web se adapte perfectamente a cualquier pantalla, desde un móvil pequeño hasta un monitor gigante.

Se enfoca en lo visual y lo que el usuario toca (botones, iconos, espaciado).

Se enfoca en cómo se siente el usuario. Busca que la navegación sea lógica, fácil y satisfactoria.

3. Herramientas y Frameworks

Conjuntos de herramientas y reglas predefinidas que ayudan a los programadores a trabajar de forma más rápida y organizada.

Framework robusto mantenido por Google, ideal para crear aplicaciones web empresariales de gran escala.

Librería creada por Meta (Facebook) que permite construir interfaces de usuario basadas en componentes reutilizables.

Framework progresivo de JavaScript muy fácil de aprender, diseñado para ser adaptable y ligero.

El framework de CSS más popular que ofrece componentes listos (como este acordeón) para agilizar el diseño web.

Framework de CSS basado en clases de utilidad, lo que permite diseñar interfaces personalizadas directamente en el HTML.

Un framework de diseño profesional y avanzado, muy enfocado en la flexibilidad y el diseño responsivo.

4. Tecnologías del Lado del Servidor (Backend)

Lenguaje de programación versátil y fácil de leer, muy utilizado para lógica de servidor, datos e IA.

Lenguaje clásico diseñado específicamente para el desarrollo web del lado del servidor.

Entorno que permite ejecutar JavaScript fuera del navegador, específicamente en el servidor.

Plataforma de desarrollo de Microsoft para crear aplicaciones empresariales potentes y seguras.

5. Datos y Comunicación

Bases de datos relacionales que organizan la información en tablas fijas (como filas y columnas).

Bases de datos no relacionales, ideales para manejar grandes volúmenes de datos con estructuras flexibles.

Conjunto de reglas que permiten que dos aplicaciones se comuniquen y compartan datos entre sí.

6. Infraestructura y Publicación

Computadoras potentes conectadas a internet que guardan y envían los archivos de tu web a los usuarios.

El servicio de alquiler de espacio en un servidor para que tu sitio web sea accesible en todo el mundo.

La dirección única y fácil de recordar que los usuarios escriben para visitar tu web (ej: misitio.com).

Red de servidores globales que entregan el contenido de tu web desde el servidor más cercano al usuario para mayor velocidad.

7. Gestión y Estrategia

Software que permite crear y administrar contenidos de una web sin necesidad de escribir código (ej: WordPress).

Técnicas utilizadas para mejorar la posición de tu página en los resultados de buscadores como Google.
